INSTALLATION
pour un bon fonctionnement du programme, les fichiers "contour_france.csv" , "play" et "pause" doivent être dans le répertoire d'où est lancé le programme (généralement le répertoire de l'exécutable)
De même, les fichiers "aerodromes_fr.txt" "balises_fr.txt" et "PDV.txt" doivent être dans ce même répertoire pour que la fonction "chargement par défaut" fonctionne.

Pour exécuter le programme, lancez le fichier "GATI".
